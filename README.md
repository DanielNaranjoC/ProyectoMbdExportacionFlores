# ProyectoMbdExportacionFlores
Proyecto de análisis y modelado de series temporales para las exportaciones ecuatorianas de flores naturales. Incluye limpieza y validación de datos, generación de gráficos exploratorios, pruebas de estacionariedad, modelos ETS, Prophet, SARIMAX y SARIMAX-GARCH, además de comparación de predicciones mediante métricas de error.
